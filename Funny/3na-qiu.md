###框内有100个球，两个人分别轮流拿，最少拿1个，最多5个，谁先拿到100个算赢。规定你先拿，你第一次拿几个，能保证你一定赢？

假设我赢了，我拿的序列是
```
A0,A1,A2,A3...An
```
ta拿的序列是
```
B1,B2,B3...Bn
```

那么
```
A0+(A1+B1)+(A2+B2)+(A3+B3)+...+(An+Bn)=100
```
那么我可以保证，后面的每个括号里的和都是6，ta拿1个我拿5个，ta拿5个我拿1个。所以
```
A0+n*6=100
n=(100-A0)/6
```
n一定未正整数，而A0属于[1, 5]，所以只有A0=4，才能保证n为正整数。
