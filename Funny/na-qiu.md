### 问：框内有100个球，两个人轮流拿，最少1个球，最多拿5个，谁先拿到第100个谁就算赢。规定你先拿，你第一次拿几个才能保证你一定赢？

假设我赢了，我拿球的序列是
```
A0,A1,A2,A3...An
```
ta拿的序列是
```
B1,B2,B3...Bn
```
那么

```
A0+(A1+B1)+(A2+B2)+(B3+A3)+...+(An+Bn)=100
```
我能保证的是，后面括号里的和都等与6，ta拿1个我拿5个，ta拿5个我拿1个。所以得到以下等式：

```
A0+n*6=100
n=(100-A0)/6
```
我们知道n一定是整数，而A0的取值是[1, 6]，所以只有A0=4的时候n为整数，所以第一次拿4个球才能保证我一定赢。
